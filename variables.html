파이썬 변수와 자료형

이 문서는 파이썬의 변수(assignment)와 핵심 자료형을 실전 예제와 함께 빠르게 익히도록 구성했습니다. 각 예제는 복사해 바로 실행할 수 있습니다.

1. 변수(assignment)의 개념

파이썬에서 변수는 “이름 → 객체”의 참조(reference)입니다. 선언 문법이 따로 없고, 대입과 동시에 생성됩니다.

x = 10          # 정수 객체를 x가 가리킴
y = x            # y도 같은 정수 객체를 참조
x = x + 5        # 새로운 정수(15)를 만들어 x가 그것을 참조
print(x, y)      # 15 10

1.1 변수 이름 규칙

  • 영문/숫자/밑줄 사용, 숫자로 시작 불가 (예: total_sum, _cache)
  • 대소문자 구분 (Value ≠ value)
  • 예약어 금지 (class, for, def 등)

1.2 다중 대입 & 언패킹

a, b = 1, 2
a, b = b, a            # 스왑
x, y, *rest = [10, 20, 30, 40]
print(a, b, x, y, rest)  # 2 1 10 20 [30, 40]

2. 내장 자료형 개요

💡
불변(immutable): int, float, bool, str, tuple, frozenset …
가변(mutable): list, dict, set, bytearray …

2.1 숫자형: int, float

n = 42             # int, 임의정밀도
pi = 3.14159       # float(배정밀도)
print(type(n), type(pi))
print(7 // 3, 7 % 3, 2 ** 10)  # 정수 나눗셈, 나머지, 거듭제곱

2.2 bool

flag = True
print(bool(0), bool(""), bool([]))   # False False False
print(bool(123), bool("hi"), bool([0]))  # True True True

2.3 str (문자열)

s = "Python"
print(s.lower(), s.upper(), s[0], s[-1])
print("Py" in s, s.replace("Py", "My"))
name, score = "민수", 98
print(f"{name}님의 점수는 {score}점")  # f-string

2.4 list (가변 시퀀스)

nums = [1, 2, 3]
nums.append(4)
nums.extend([5, 6])
nums[0] = 99
print(nums, nums[:3], nums[-2:])  # [99,2,3,4,5,6] [99,2,3] [5,6]

2.5 tuple (불변 시퀀스)

pt = (10, 20)
# pt[0] = 30  # TypeError (불변)
x, y = pt     # 언패킹
print(x, y)

2.6 dict (키-값 매핑)

user = {"name": "민수", "age": 20}
user["job"] = "student"
print(user.get("age"), user.keys(), user.values())

2.7 set (중복 없는 집합)

tags = {"python", "web", "python"}
print(tags)                      # {'python', 'web'}
print({"a","b"} | {"b","c"})     # 합집합
print({"a","b"} & {"b","c"})     # 교집합

2.8 None

def find_user(id): 
    return None  # 없음을 명시
print(find_user(1) is None)  # True (동일성 비교는 is)

3. 동등성(==) vs 동일성(is)

a = [1,2,3]
b = [1,2,3]
print(a == b)  # 값이 같다  -> True
print(a is b)  # 같은 객체? -> False

4. 타입 확인과 변환

print(type(3), isinstance(3, int))  # <class 'int'> True
print(int("10"), float("3.14"), str(42))

5. 가변/불변과 참조

items = [1, 2]
alias = items        # 같은 리스트 참조
alias.append(3)
print(items)         # [1, 2, 3]

# 안전한 복사
copy1 = items[:]     # 슬라이스 복사
copy2 = list(items)  # 생성자 복사

6. 흔한 함정: 가변 기본값

def append_item(x, bucket=[]):   # ❌ 안티패턴
    bucket.append(x)
    return bucket

print(append_item(1))  # [1]
print(append_item(2))  # [1, 2]  ← 이전 호출의 리스트 재사용!
✅
권장 패턴 
def append_item(x, bucket=None):
    if bucket is None:
        bucket = []
    bucket.append(x)
    return bucket

7. Truthy/Falsy와 조건식

def greet(name):
    if not name:            # "", None, [], 0 등은 False로 간주
        return "이름을 입력하세요"
    return f"안녕하세요, {name}님!"

print(greet(""))    # 이름을 입력하세요

8. 미니 퀴즈

  1. a = [1,2]; b = a; b.append(3) 후 a는 무엇일까요?
  2. x = (1, 2, [3]); x[2].append(4); x의 결과는?
  3. None 비교에 적절한 연산자는 ==일까요 is일까요?
  4. 문자열 "123"을 숫자 123으로 바꾸는 방법은?
정답 보기
  1. [1, 2, 3] (같은 리스트 참조)
  2. (1, 2, [3, 4]) (튜플은 불변이지만 내부 리스트는 가변)
  3. is (None은 단일톤 객체이므로 동일성 비교)
  4. int("123")

9. 치트시트

  • 불변: int, float, bool, str, tuple, frozenset
  • 가변: list, dict, set
  • == 값 비교, is 객체 동일성 비교
  • type(x) / isinstance(x, T) / 캐스팅 int/float/str/list …
  • 가변 기본값 금지 → None 사용 패턴
