파이썬 기초배우기

설치부터 기본 문법·자료형·제어문·함수·모듈까지 하루 한 걸음 배우는 블로그. 예제로 익히고, 짧은 퀴즈로 확인해요.

🐍 초보 환영 💡 짧고 실용적인 예제 🧪 실행 가능한 코드

시작하기

  1. 파이썬 설치(Windows/macOS)
  2. 첫 코드 실행: print("Hello, Python!")
  3. 문법 기초: 변수 · 자료형 · 연산자

예제 미리보기

# 리스트 컴프리헨션
squares = [n*n for n in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]
